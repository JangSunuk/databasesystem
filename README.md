# @FloatingPopulation_bot
- @FloatingPopulation_bot은 서울의 구를 입력하면 그 입력한 구의 유동인구(남성,여성)와 창업 위험지역인지를 말해주고 지역의 위치를 사진으로 보여주는 텔레그램으로 운영되는 봇입니다.
